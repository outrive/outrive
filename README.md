## What is OUTRIVE?

OUTRIVE is the premier launchpad for AI agents built on [Virtuals Protocol](https://app.virtuals.io), deployed on Robinhood Chain (chainId 4663). Anyone can launch a tokenized AI agent, stake, and participate in a transparent bonding curve market.

## Repositories

| Repo | Description |
|---|---|
| [app](https://github.com/outrive/app) | Full-stack web application — React + Vite frontend, Express API server |
| [contracts](https://github.com/outrive/contracts) | Contract ABIs and on-chain configuration for Robinhood Chain |
| [sdk](https://github.com/outrive/sdk) | TypeScript SDK for the OUTRIVE API *(coming soon)* |

## Stack

- **Frontend** — React 18, TypeScript, Vite, Tailwind CSS, shadcn/ui
- **Blockchain** — Viem, Wagmi, Virtuals Protocol, Robinhood Chain
- **Backend** — Node.js, Express, Drizzle ORM, PostgreSQL
- **AI** — Anthropic Claude (agent simulation & chat)
